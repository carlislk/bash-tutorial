# bash-tutorial
Interactive Bash Tutorial

This is a simple bash/unix tutorial.

To run 

-> ./bash_tutorial.sh

Make sure that bash_tutorial.sh is executable.

If not you can make it executable by running. 

chmod u+x bash_tutorial.sh

Enjoy.
